# Modernized Areo

A slick, modern Aero-inspired (duh) theme that dynamically fills with your active system accent color. It enhances the taskbar background, borders, tray overflow menu, and icon geometry without breaking native functionality.

## Preview
![Screenshot](screenshot.png)

## How to use
Copy the contents of [theme.txt](theme.txt) and import it into the Windows 11 Taskbar Styler mod settings.
