# FluentGlass

A Fluent Design-inspired glass taskbar theme for Windows 11 using Windhawk.

FluentGlass blends transparency, blur, and subtle motion-inspired visuals to create a clean and modern taskbar experience that feels native to Windows 11.

## рҹӣ  Configuration

You can customize the theme via:

| Variable            | Description        |
| ------------------- | ------------------ |
| `TaskbarFrameWidth` | Taskbar width      |
| `TaskbarHeight`     | Taskbar height     |
| `GlobalRadius`      | Corner roundness   |
| `BlurAmount`        | Glass strength     |
| `TintOpacity`       | Transparency level |
| `SystemAccentColor` | Used in highlights |

---

## рҹҺЁ Customization Tips
